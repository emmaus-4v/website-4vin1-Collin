Je hebt vast al wel gehoord van de game Genshin Impact. Een gratis openworld game dat wordt gespeeld door miljoenen mensen. Genshin Impact werd gelaunched op September 28, 2020 en is ontwikkeld door de mensen van miHoYo. De game is te splen op: Android, iOS, Windows, PS4 en sinds April 28, 2021 ook op PS5. De game wordt ook developt voor Nintendo Switch en Xbox alleen hebben de makers van de game geen datum gegeven voor wanneer de game ook op deze platformen de games gespeeld kan gaan worden gespeeld.

Genshin Impact lijkt een beetje op de game: Zelda Breath of the Wild dus als je dze game leuk vond om te spelen is dit zeker een game voor jou.
